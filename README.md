The project is used for healthcare data annotation, database storage, and analysis

 ## Features
 - Create SQLite healthcare database
 - store patient records
 - Export data to excel
 - Annotation file support

 ## Files

 - main.py > Main execution file
 - create_database.py > create SQLite database
 - Database / > Database folder
 - Annotation_file/ > Annotation input file
 - output/ > Generated output files
   
 ## Technologies
 - Python
 - SQLite
 - Pandas
 - Git & Github
